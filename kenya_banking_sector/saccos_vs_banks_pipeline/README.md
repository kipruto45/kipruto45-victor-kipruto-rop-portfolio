# 🚜 SACCOs vs. Banks Pipeline

![Status](https://img.shields.io/badge/Status-Roadmap-blue)

## 📋 Overview
A unique data pipeline comparing the financial performance of Kenyan SACCOs (Savings and Credit Co-operatives) against commercial banks. This project aims to track the competitive overlap in the Kenyan micro-lending space.

## 🚀 Planned Integrations
- **SASRA Data**: Ingesting regulatory filings from the SACCO Societies Regulatory Authority.
- **Market Overlap**: Analyzing where SACCOs are outperforming banks in rural deposits.

---
*Developed as part of the Kenya Banking Sector Analytics Portfolio*
