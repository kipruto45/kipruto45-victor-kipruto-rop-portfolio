# 🤝 NCBA Post-Merger Analytics

![Status](https://img.shields.io/badge/Status-Case_Study-purple)

## 📋 Overview
A case-study analytics project tracking the performance of NCBA Bank post-merger (NIC & CBA). Examines synergy realization, cost-savings, and the growth of the digital-only M-Shwari product.

---
*Developed as part of the Kenya Banking Sector Analytics Portfolio*
