# 💸 Interest Rate Spread Analyzer

![Status](https://img.shields.io/badge/Status-Roadmap-blue)

## 📋 Overview
Monitoring the "Spread" between deposit rates and lending rates across the Kenyan banking sector. Provides insights into bank margins and the impact of CBR (Central Bank Rate) changes.

---
*Developed as part of the Kenya Banking Sector Analytics Portfolio*
