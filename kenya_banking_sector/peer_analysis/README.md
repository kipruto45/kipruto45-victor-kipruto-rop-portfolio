# 📊 Peer Analysis & Normalization

![Status](https://img.shields.io/badge/Status-Implemented-green)

## 📋 Overview
Automated benchmarking for Kenya's mid-tier banks (Stanbic, StanChart, DTB, I&M). This project uses Z-Score normalization to provide a fair comparison between institutions of different scales.

## 🚀 Key Features
- **Z-Score Normalization**: Removing scale bias to compare efficiency.
- **Efficiency Frontier**: Visualizing the trade-off between Scale and Profitability.
- **Automated Commentary**: LLM-ready data structures for generating investor-style analysis.

*Developed as part of the Kenya Banking Sector Analytics Portfolio*
